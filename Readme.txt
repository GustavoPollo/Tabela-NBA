Pegar da tabela do site os valores da tabela de cada time e suas informações e salvar 
em um arquivo .json. Após rodar a primeira vez o programa deverá perguntar ao usuário 
se ele irá querer atualizar os dados, além de dar ao usuário filtros sobre os times,como 
quantidade de pontos, quantas vitórias/derrotas entre outros. Caso o usuário queira 
atualizar os dados, o programa deverá perguntar se é algo especifico, como apenas de um time/times, 
conferencia ou tudo e ao final sobrescrever os valores no arquivo .json
